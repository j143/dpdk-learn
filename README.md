# dpdk-learn

## React App

A Vite + React scaffold for experimenting with React.

### Paste your React code

Open **`react-app/src/App.jsx`** and replace its contents with your own React component.

### Run locally

```bash
cd react-app
npm install
npm run dev
```

### GitHub Pages deployment

Every push to `main` that touches files inside `react-app/` automatically builds the app and deploys it to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.

The site will be available at:  
`https://j143.github.io/dpdk-learn/`

> **First-time setup:** In your repository **Settings → Pages**, set the source to **"GitHub Actions"**.
